# Overturn

My first game ever made.

This was inspired by Undertale (thank you Toby!).

Built using Python without any game engine.

This project includes:

- Multiple endings
- Save system (personally I am proud of this)
- Lore-driven gameplay
- A game Icon

It has bugs and unfinished parts, but I decided to leave it as it is
as a memory of where I started.

This project is only a prototype, and there may be more to it in the future.

## Notes

The secret ending was coded but never actually implemented in the game due to a missed function call.

## Endings

- Massacre Ending (kill everyone)
- True Ending (spare everyone except Emru)
- Pacifist Ending (spare everyone)
- Puppet Ending (kill everyone except Emru)
- Killer Ending (kill more NPCs than you spare)
- Dark Heart Ending (spare more NPCs than you kill)
- Secret Ending (not triggered due to a missing function call)
- Sus Ending (type "sus" when prompted for a name)
